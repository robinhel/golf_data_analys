# Golf data analys – PGA Tour 2010–2018

Jag spelar golf och har alltid undrat vad som egentligen gör skillnad. Ska man lägga tid på drivern eller putting greenen? Det här datasetet gav mig ett sätt att faktiskt kolla på den frågan med riktig data, även om svaret gäller världens bästa spelare och inte min handicap.

Analys av PGA Tour-statistik som en del av kursen MAI25HA.

## Dataset

Hämtat från Kaggle: [PGA Tour Golf Data](https://www.kaggle.com/datasets/jmpark746/pga-tour-data-2010-2018)  
Lägg filen `pgaTourData.csv` i samma mapp som notebooken innan du kör.

## Köra notebooken

jupyter notebook Golf_analys_kk1.ipynb
